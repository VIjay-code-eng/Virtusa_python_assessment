# My Smart Expense Tracker

I built this terminal-based expense tracker to learn and practice Python. It's a simple program that helps the customer to keep track of daily expenses by saving them into a CSV file. It also helps to view a summary of customer spending habits to see where the money is going. 

## What it does

* Lets the customer add new expenses with the amount, category, and date.
* Saves everything into a `customer_expenses.csv` file so data isn't lost when the program closes.
* Shows a list of all the expenses customer added.
* Gives the customer a total summary of all the spending.
* Breaks down the spending by category so the customer can see what they spend the most money on.

## Technologies I used

* Just core Python! No external libraries required.
* Basic File handling for saving into CSV format.
* Python dictionaries and while loops to keep the menu running and calculate my totals.

## How to use it

1. Open up the project folder.
2. Open your terminal in this folder.
3. Run the python script:

```
python main.py
```
